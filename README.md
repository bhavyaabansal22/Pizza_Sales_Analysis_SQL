# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview

This project analyzes a pizza restaurant's sales data using SQL to uncover valuable business insights related to customer ordering behavior, revenue generation, product performance, and sales trends. The project demonstrates the practical application of SQL in solving real-world business problems through data exploration and analysis.

---

## 📂 Dataset

The project uses four related tables:

- Orders
- Order Details
- Pizzas
- Pizza Types

These tables contain information about customer orders, pizza sizes, categories, prices, and quantities sold.

---

## 🛠 Skills Demonstrated

- SQL Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Subqueries
- Date & Time Functions
- Business Data Analysis

---

## 📊 Business Questions Answered

- Retrieve the total number of orders placed.
- Calculate total revenue generated.
- Identify the highest-priced pizza.
- Find the most commonly ordered pizza size.
- List the top 5 most ordered pizza types.
- Calculate the total quantity sold for each pizza category.
- Analyze the distribution of orders by hour.
- Determine the category-wise distribution of pizza types.
- Calculate the average number of pizzas ordered per day.
- Identify the top 3 pizza types based on revenue.

## 📈 Key Insights

- Calculated the overall sales revenue generated by the business.
- Identified premium pizzas with the highest selling price.
- Determined customers' preferred pizza size.
- Found the best-selling pizza varieties based on quantity.
- Analyzed customer ordering trends throughout the day.
- Compared demand across pizza categories.
- Estimated the average daily pizza sales.
- Ranked pizza types based on revenue contribution.
